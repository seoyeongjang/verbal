# 개인정보처리방침 초안

원문: `docs/PRIVACY_POLICY.md`

공개 URL: `https://verbal.chat/privacy`

Verbal은 인증, 메시징, 음성 변환, 안전, 알림, 서비스 운영에 필요한
최소 데이터를 처리합니다.

## 처리 데이터

- 전화번호 인증 식별자.
- 사용자 아이디/handle과 표시 이름.
- 메시지, 음성파일, transcript, 첨부파일, 공유 위치.
- 방 멤버십, 초대 링크, 신고, 차단, 안전 관련 메타데이터.
- 푸시 알림 토큰.
- 비용, 안정성, 악용 모니터링을 위한 사용량 진단 정보.

## 음성과 STT

음성 메시지는 STT로 텍스트 변환됩니다. 프로덕션에서는 Deepgram이 STT 처리를
위해 음성을 수신할 수 있습니다. 음성 보존기간은 방 단위이며 기본값은 1일입니다.
만료 후 음성파일은 삭제되고 transcript 텍스트는 남습니다.

## 사용자 통제

- 보낸 메시지 삭제.
- 권한이 있는 방의 음성 보존기간 변경.
- 데이터 및 저장 공간에서 계정 데이터 내보내기.
- 메뉴 -> 계정 관리에서 계정 삭제.
- 방 또는 메시지 신고.
- 사용자 차단.

## 공유

앱 운영에 필요한 범위에서 Firebase/Google Cloud와 STT용 Deepgram 등 인프라
제공자에게 데이터가 처리될 수 있습니다.

## 문의

개인정보와 데이터 삭제 문의는 `support@verbal.chat`으로 보낼 수 있습니다.
