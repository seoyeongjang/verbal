# 고객지원 매크로

원문: `docs/SUPPORT_MACROS.md`

## 로그인

먼저 전화번호 인증 상태를 확인하겠습니다. 전화번호, 국가번호, 기기 모델, OS
버전, 인증 실패 시간을 알려주세요.

## STT 실패

음성 메시지는 STT 성공 후 자동 전송됩니다. STT가 실패하면 앱에서 STT 재시도
또는 transcript 직접 입력 후 전송할 수 있습니다. 녹음 시간, 사용 언어, 기기,
복구 시트에 표시된 오류 문구를 알려주세요.

## 계정 분실

계정은 전화번호 인증과 사용자 아이디에 연결됩니다. 국가번호가 포함된 전화번호,
사용자 아이디, 마지막 로그인 날짜를 알려주세요.

## 신고 처리

신고는 필요한 최소한의 메시지 또는 방 맥락만 확인해 검토합니다. 심각도에 따라
메시지, 방, 계정 단위의 조치를 취할 수 있습니다.

## 데이터 삭제

메뉴 -> 계정 관리 -> 계정 삭제에서 프로필 데이터, 아이디 선점, 푸시 토큰,
활성 대화방 멤버십을 삭제할 수 있습니다. 기존 전송 메시지는 계정 삭제 전
개별 삭제하지 않았다면 대화 맥락 유지를 위해 익명화됩니다.
