# 제품 계획

## 포지셔닝

Verbal은 한국 일상 대화에서 말하기를 가장 빠른 입력 방식으로 쓰는 모바일 메신저입니다. 사용자는 짧은 음성 메시지를 녹음하고, 앱은 이를 텍스트로 변환하며, 수신자는 텍스트를 읽거나 원본 음성을 재생할 수 있습니다.

## MVP 범위

- 전화번호 로그인과 프로필 설정.
- 핸들 기반 1:1 및 소규모 그룹 대화 초대.
- Firestore 실시간 방 목록과 메시지 목록.
- 텍스트 메시지 전송.
- 음성 메시지 녹음.
- 자동 전송: 녹음, STT 완료 후 확인 시트 없이 음성+텍스트 전송.
- STT 복구: STT 실패 시 재시도 또는 transcript 직접 입력 후 전송.
- 초대 링크/QR, 그룹 관리자 권한, 첨부/위치, 예약 전송, 번역.
- 기본 음성파일 1일 보존과 방별 보존기간 설정.
- 텍스트/음성 사용량과 음성 메시지 길이는 무제한.
- 메시지 삭제와 신고 함수 엔드포인트.

## MVP 제외 범위

- 개인 메신저 구독 과금.
- 개인 채팅방 내부 광고 삽입.
- 데스크톱 앱, 오픈채팅 탐색, 완전한 종단간 암호화.
- 말하는 중 실시간 자막. 백엔드는 추후 Realtime transcription을 붙일 수 있게 유지합니다.
- 신고 접수 이상의 대규모 커뮤니티 운영 워크플로.

## 성공 지표

- 음성 메시지 전송 성공률.
- 평균 STT 지연시간.
- STT 복구율.
- 음성 재생률.
- 기본 1일 음성 보존 정책 채택률.
- DAU당 STT 비용과 STT 캐시 적중률.
- MAU당 광고 매출 및 B2B/커머스 매출.
