# 부하와 비용 시뮬레이션

원문: `docs/LOAD_COST_SIMULATION.md`

## 현재 정책

- 정상 텍스트 사용량: 무제한.
- 정상 음성 개수: 무제한.
- 음성 길이: 제품 레벨의 고정 제한 없음.
- 악용 방지는 초대 생성, 초대 참여, 신고, 차단 등 민감 액션에만 적용합니다.

## 추적 변수

- MAU와 DAU.
- DAU당 음성 메시지 수.
- 평균 음성 길이(초).
- STT 성공률과 재시도율.
- STT 캐시 hit rate.
- 음성 보존기간.
- 음성 재생률.
- 첨부 개수와 평균 크기.
- 활성 세션당 Firestore 읽기/쓰기.

## 계산식

월간 STT 분:

`DAU * DAU당_음성수 * 평균_음성초 * 30 / 60`

월간 활성 음성 저장량:

`DAU * DAU당_음성수 * 평균_음성_MB * 보존일`

## 런칭 전 필수

- 공개 트래픽 전 베타 telemetry로 추정치를 다시 계산합니다.
- Firebase/GCP 예산과 Deepgram 사용량 알림을 설정합니다.
- 정상 사용자 사용량과 이상 급증 사용량을 분리해 모니터링합니다.
