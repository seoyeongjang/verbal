# 베타 QA 계획

원문: `docs/BETA_QA_PLAN.md`

## 기기 매트릭스

- 소형 Android 폰.
- 대형 Android 폰.
- 저사양 Android 기기.
- Android 13 이상 알림 권한 기기.
- 느린 네트워크와 재연결 상황.
- iOS 런칭 범위에 포함된다면 iOS/TestFlight 기기.

## 핵심 시나리오

- 전화번호 인증, 프로필 설정, 아이디 검증.
- 1:1 방과 그룹 방 생성.
- 메시지 전송, 수정, 삭제, 답장, 반응, 고정, 검색.
- 확인 후 전송 모드와 즉시 전송 모드의 음성 녹음.
- STT 실패 복구: STT 재시도와 직접 transcript 입력 전송.
- 이미지, 파일, 위치 전송.
- 기본 preset 없는 예약 메시지 전송.
- 메시지 번역.
- 초대 링크/QR 생성, 참여, 승인, 거절, 폐기.
- 그룹 역할 변경, 멤버 내보내기, 방 나가기.
- 메시지 신고, 방 신고, 사용자 차단.
- 내 데이터 내보내기와 계정 삭제.

## 통과 기준

- 깨진 한글 텍스트가 없습니다.
- 360px 폭에서 버튼 또는 말풍선 텍스트가 잘리지 않습니다.
- 음성 보존기간 만료 후에도 transcript가 남습니다.
- 실패한 전송을 앱 재시작 없이 다시 시도할 수 있습니다.
- 음성과 첨부 전송 중 명확한 진행 상태가 표시됩니다.
- 개인 채팅방 내부에는 광고가 표시되지 않습니다.
