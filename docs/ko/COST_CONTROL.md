# 원가 절감 및 수익화 정책

Voice Messenger는 무료 개인 메신저입니다. 개인 채팅 구독 과금은 핵심 비즈니스 모델에서 제외합니다.

## 기본값

- 사용자 메시징 사용량은 무제한입니다. 텍스트/음성 건수는 비용 모니터링 목적으로만 집계합니다.
- 음성 메시지 길이는 무제한입니다. 0.5초 미만의 비어 있거나 너무 짧은 녹음만 거절합니다.
- 음성 녹음 품질: 지원되는 환경에서 16 kHz 샘플레이트, 64 kbps 목표 비트레이트.
- 채팅방 기본 음성파일 보존기간: 1일.
- 보존기간 옵션: 1일, 7일, 사용자 지정 1-30일.
- 만료 후에는 음성파일만 삭제하고 transcript는 Firestore에 유지합니다.

## 백엔드 강제 정책

- Cloud Functions는 `usageDaily/{uid}_{yyyy-mm-dd}`를 Asia/Seoul 날짜 기준으로 기록하지만, 일일 건수로 메시지를 거절하지 않습니다.
- STT 비용은 메시지 최종 전송 시점이 아니라 transcription 요청 시점에 집계합니다.
- 음성파일은 SHA-256 해시로 중복 여부를 판단하고 `transcriptionCache`를 사용해 Deepgram 중복 호출을 줄입니다.
- `expireVoiceAudio`가 매시간 만료된 메시지/초안 음성파일을 삭제합니다.
- Storage rules는 음성 업로드에 고정 용량 제한을 두지 않습니다. Functions는 0.5초 미만의 비어 있거나 너무 짧은 녹음만 거절합니다.

## 수익화 지면

- 네이티브 광고는 DM 목록, 채널 목록, 요청함, 노트 주변 영역에만 배치합니다.
- 개인 채팅 메시지 타임라인 내부에는 광고를 삽입하지 않습니다.
- 추가 수익원은 공식계정, 비즈니스 메시징, 캠페인 알림, 예약/주문 알림, 쿠폰, 로컬딜, 커머스 제휴 수수료로 설계합니다.
- 기업용 STT/요약은 opt-in 된 비즈니스 채널에만 제공합니다.

## 핵심 KPI

- DAU당 STT 분.
- DAU당 음성 메시지 수.
- 평균 음성 길이.
- STT 캐시 적중률.
- STT 실패율.
- 음성파일 삭제 성공률.
- 보존기간별 활성 음성 저장량.
- MAU당 광고 ARPU와 B2B/커머스 매출.
