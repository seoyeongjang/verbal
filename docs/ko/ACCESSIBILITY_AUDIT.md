# 접근성 점검

원문: `docs/ACCESSIBILITY_AUDIT.md`

## 현재 반영 사항

- 주요 아이콘 전용 버튼에 tooltip을 제공합니다.
- 메시지 말풍선은 텍스트가 잘리지 않도록 너비를 제한합니다.
- composer 버튼은 안정적인 터치 영역을 유지합니다.
- 오류, 재시도, 전송 중 상태는 색상만이 아니라 텍스트로도 표시합니다.
- 초록색 팔레트는 대비 확보를 위해 검정/흰색/회색 텍스트와 함께 사용합니다.

## 수동 QA

- 텍스트 확대 100%, 130%, 160%에서 확인합니다.
- 360px 폭에서 한국어 라벨이 잘리지 않는지 확인합니다.
- 스크린리더 포커스 순서: 홈 탭, 방 목록, composer, 메시지 액션 시트, 대화
  정보.
- 저사양 Android 기기에서 터치 영역을 확인합니다.
- 모든 파괴적 액션에 명확한 라벨이 있는지 확인합니다.

## 후속 작업

기기 베타 중 TalkBack/VoiceOver 실제 점검을 수행합니다. 자동 점검만으로는 채팅
메시지 읽기 순서와 bottom sheet 상호작용 품질을 완전히 검증할 수 없습니다.
