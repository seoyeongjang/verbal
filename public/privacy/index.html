개인정보처리방침

Verbal은 인증, 메시징, 음성 STT, 안전, 알림, 서비스 운영에 필요한 최소한의 데이터를 처리합니다.

처리하는 데이터

  • 전화번호 인증 식별자
  • 사용자 아이디, 표시 이름, 프로필 정보
  • 메시지, 음성 파일, transcript, 첨부파일, 공유 위치
  • 대화방 멤버십, 초대 링크, 신고, 차단, 안전 metadata
  • 푸시 알림 토큰과 서비스 안정성/비용 관리를 위한 사용 진단 정보

음성 및 STT

음성 메시지는 텍스트 변환을 위해 STT로 처리됩니다. 운영 환경에서는 Deepgram이 음성 전사를 위해 오디오를 처리할 수 있습니다. 음성 파일은 방 단위 보존기간을 따르며 기본값은 1일입니다. 만료 후 음성 파일은 삭제되고 transcript는 대화 기록과 검색을 위해 남을 수 있습니다.

사용자 제어

  • 보낸 메시지 삭제
  • 권한이 있는 방의 음성 보존기간 변경
  • 데이터 및 저장 공간 메뉴에서 계정 데이터 내보내기
  • 메뉴의 계정 관리에서 계정 삭제 요청
  • 대화방 또는 메시지 신고, 사용자 차단

제3자 처리

Firebase/Google Cloud는 인증, 데이터베이스, 스토리지, Functions, 푸시, 로깅, 모니터링 인프라를 제공합니다. Deepgram은 STT 기능을 위해 음성 오디오를 처리할 수 있습니다.

문의

개인정보와 데이터 삭제 문의는 support@verbal.chat으로 보낼 수 있습니다.
