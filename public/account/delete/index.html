Verbal account deletion

계정 삭제 요청

Verbal 계정은 앱 안에서 직접 삭제할 수 있으며, 앱에 접근할 수 없는 경우 이 페이지의 안내에 따라 삭제 요청을 보낼 수 있습니다.

앱에서 삭제하는 방법

  1. Verbal 앱을 엽니다.
  2. 홈 화면의 메뉴를 엽니다.
  3. 계정 관리로 이동합니다.
  4. 계정 삭제를 선택합니다.
  5. 삭제 안내를 확인한 뒤 요청을 완료합니다.

웹에서 삭제 요청하기

앱에 로그인할 수 없다면 아래 이메일 요청 버튼을 사용해 주세요. 본인 확인을 위해 가입 전화번호, 사용자 아이디, 삭제 요청 사유를 함께 보내야 할 수 있습니다.

이메일로 삭제 요청 데이터 삭제 정책

삭제되는 데이터

  • 프로필 표시 정보, 아이디 선점, 푸시 토큰
  • 활성 대화방 멤버십
  • Firebase Auth 계정

보존될 수 있는 데이터

이미 상대방에게 전송된 메시지는 대화 맥락 유지를 위해 보존될 수 있습니다. 이 경우 발신자는 삭제된 계정으로 익명화됩니다. 음성 파일은 방의 보존기간 정책에 따라 삭제되며 transcript는 대화 기능을 위해 남을 수 있습니다.

삭제 처리는 본인 확인과 법적 보존 의무 확인 후 진행됩니다. 요청 상태와 추가 확인이 필요한 경우 `support@verbal.chat`으로 안내합니다.
