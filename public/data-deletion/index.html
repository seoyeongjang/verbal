데이터 삭제 정책

이 페이지는 Verbal 계정 삭제와 데이터 삭제 요청 시 처리되는 데이터의 범위를 설명합니다.

삭제 또는 비활성화되는 데이터

  • 사용자 프로필 표시 정보
  • 사용자 아이디 선점 정보
  • 푸시 알림 토큰
  • 활성 대화방 멤버십
  • Firebase Auth 계정

보존될 수 있는 데이터

  • 상대방에게 이미 전달된 메시지와 대화 맥락
  • 법적 의무 또는 운영상 필요한 신고/보안 기록
  • 음성 파일 삭제 후에도 대화 검색과 접근성을 위해 필요한 transcript

음성 파일 보존기간

음성 파일은 대화방의 보존기간 설정을 따릅니다. 기본 보존기간은 1일이며, 만료 후 음성 파일은 삭제되고 transcript는 유지될 수 있습니다.

요청 경로

앱에서 직접 삭제하려면 Verbal 앱의 메뉴 -> 계정 관리 -> 계정 삭제를 사용하세요. 앱에 접근할 수 없다면 웹 삭제 요청 페이지를 이용하세요.

계정 삭제 요청으로 이동

앱에 접근할 수 없거나 추가 확인이 필요하면 support@verbal.chat으로 데이터 삭제 문의를 보낼 수 있습니다.

데이터 삭제 요청은 본인 확인과 법적 보존 의무 확인 후 처리됩니다. 추가 확인이 필요한 경우 `support@verbal.chat`으로 안내합니다.
